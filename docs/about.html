About Section 508 Pro

Section 508 Pro is a best-of-breed, open source digital accessibility dashboard. As a part of a full "human-in-the-loop" solution, it helps identify accessibility issues in your web pages and documents so that designers, testers, and developers can quickly fix them.

You can download Section 508 Pro and set it up on your own. Or ThirstyHead -- the creator and maintainer of Section 508 Pro -- can help install it in your organization as a part of a full Accessibility Organizational Change Management program.

Technology

Section 508 Pro is built using free, industry-leading open source solutions. This means that you can focus your budget dollars on solving accessibility problems, not paying for software licenses.

Lighthouse

Lighthouse is an open-source, automated tool from Google for improving the quality of web pages.

If you have Google Chrome installed, you already have Lighthouse installed as well! Simply visit a web page, right-click anywhere on the page, and choose 'Inspect'. From there, choose Lighthouse and run an audit to get advice on how to improve the web page's performance, accessibility, SEO (Search Engine Optimization), and more.

Section 508 Pro has already run Lighthouse audits on thousands of US government web pages so you don't have to. But if you ever question the results you see on our website, you can run a Lighthouse audit yourself to verify. That's the beauty of open source solutions.

For another example of using Lighthouse for accessibility audits, check out the Web Almanac chapter on Accessibility. You can read a detailed analysis of the "Accessibility State of the Union" on over 8.5 million worldwide public web pages.

US Web Design System

The US Web Design System makes it easier to build accessibile, mobile-friendly US government websites. Starting with an accessible foundation like the USWDS is the first step towards making Universal Design core to your organization's principles.
